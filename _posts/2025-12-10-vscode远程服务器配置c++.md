# CUDA Debug

### 可以用vscode自带的shift +f5
tasks.json这么写
```c
"type": "cppbuild",

"label": "nvcc: build active file",

"command": "/usr/local/cuda-11.1/bin/nvcc",

"args": [

"-g",

"${file}",

"-o",

"${fileDirname}/${fileBasenameNoExtension}"

],
```


关键是换个编译器，nvcc就行。cuda-gdb不用，貌似可以用gdb debug cuda代码
